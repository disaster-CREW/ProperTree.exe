# ProperTree.exe - (v4 Stable)

This is a custom, professional distribution of **ProperTree**, focused on stability and ease of use. This version uses the **v4 source code** to avoid the bugs and pathing issues found in newer releases.

## 🚀 Features
- **Professional Installer:** A clean "Wizard" setup experience.
- **Custom Icon:** Features a high-quality custom look for the app and shortcuts.
- **Stable Core:** Built from the v4 source to ensure it works every time.

## 🛠️ How to Install
1. Download the `ProperTree_Setup.exe` from the **Releases** section.
2. Run the installer (Administrator privileges required for file association).
3. Enjoy a stable .plist editing experience!

### 🛡️ Why does Windows flag the installer?
Because this is a custom build, it is not "signed" with a paid Microsoft developer certificate. 

**To continue:**
1. Click **"More info"** on the blue SmartScreen window.
2. Click **"Run anyway."**
